# Pull Request

## 1 Styling code

- TypeScript check

```bash
yarn type-check
```

- Eslint check

```bash
yarn lint
```

- Format check

```bash
yarn format
```

## 2 Testing

- Vitest

```bash
yarn test:vitest
```

- Playwright

```bash
yarn test:playwright
```

- Build

```bash
yarn build
```

# Deployment

## Staging

- Testing workflow

```bash
yarn deploy:staging
```

## Production

- Testing workflow

```bash
yarn deploy:production
```
